Passos para instalar o PHP e MySQL nas máquinas
1. Instalar o xampp executando o programa xampp-win32-1.7.7-VC9-installer.exe
2. Com o botão direito do mouse clicar na pasta c:\xamp de escolher segurança
3. Para todos os nomes de grupos de usuários que aparecerem dar a permissão de controle total
4. No painel do xampp ligar o PHP e o MYSQL


5. copiar o arquivo agenda201102.sql para o diretório c:\xampp\mysql\bin
6. cmd no iniciar do windows
7. cd\
8. cd xampp\mysql\bin
9. mysql -u root
10. create database agenda201102;
11. exit
12. mysql -u root agenda201102 < agenda201102.sql


13. Copiar a pasta agenda do pendrive para c:\xampp\htdocs 
14. abrir um navegador
